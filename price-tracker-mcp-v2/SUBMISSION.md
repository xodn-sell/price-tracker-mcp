## 💡 프로젝트 개요

네이버 쇼핑에서 **실시간 가격 비교**, **가격 알림 설정**, **최저가 추천**을 제공하는 MCP 서버입니다.

### 핵심 가치
- 💰 **실용성:** 실제로 돈을 절약할 수 있는 서비스
- 🎯 **정확성:** 네이버 공식 API 사용으로 안정적인 데이터
- 🚀 **편의성:** 자연어로 간단하게 가격 비교

---

## 🛠️ 주요 기능 (7개 MCP 도구)

### 1. **search_product** - 상품 검색
```
입력: 키워드, 검색 개수
출력: 상품 목록 (가격, 링크, 이미지 등)
예시: "무선이어폰" → 10개 상품 목록
```

### 2. **compare_prices** - 가격 비교
```
입력: 키워드
출력: 최저가, 최고가, 평균가, 상위 10개 상품
예시: "아이폰 15" → 최저가 1,200,000원
```

### 3. **set_price_alert** - 가격 알림
```
입력: 키워드, 목표 가격
출력: 알림 설정 완료
예시: "갤럭시 버즈", 100,000원 → 알림 등록
```

### 4. **get_price_history** - 가격 히스토리
```
입력: 키워드, 조회 기간
출력: 가격 변동 기록
예시: "맥북", 30일 → 30일간 가격 추이
```

### 5. **track_product** - 상품 추적
```
입력: 키워드
출력: 추적 시작 확인
예시: "플레이스테이션 5" → 추적 등록
```

### 6. **list_tracked_products** - 추적 목록
```
입력: 없음
출력: 추적 중인 모든 상품
예시: 5개 상품 추적 중
```

### 7. **get_best_deals** - 베스트 딜
```
입력: 추천 개수
출력: 가격 대비 가치가 높은 상품
예시: 10개 베스트 딜 추천
```

### 코드 통계
```

- server.py: 170줄 (7개 도구)
- price_tracker.py: 270줄 (핵심 로직)
```
